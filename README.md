CNOC-Sans 是在我个人的兴趣下制作的，本字体希望能在符合观感的图形中，（在保证能够被正常分辨的同时）融入日常书写习惯，希望能给用户感舒适感。

本字体使用 [FontForge](https://github.com/fontforge/fontforge) 制作

## 字符完成进度
| 文字 | 完成情况（按预计完成顺序排序） |
|:-:|:-:|
| 拉丁字母   | 仅覆盖一半的英语字母 |
| 标点符号   | 少量 |
| 数字       | 少量 |
| 希腊字母   | ❌ |
| 汉字       | ❌ |
| 日文假名   | ❌ |
